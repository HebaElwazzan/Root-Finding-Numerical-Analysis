# Root-Finding-Numerical-Analysis
## Project 1 for Numerical Analysis course
This project aims to implement 5 root finding methods.
### A. Closed Methods:
1. Bisection Method
2. Regula-Falsi Method
### B. Open Methods:
1. Fixed Point Method 
2. Newton-Raphson Method
3. Secant Method
